This is the `arcs_cohort_rviz` ROS 2 package, part of the [CoHORT](https://github.com/csun-arcs/arcs_cohort) multi-rover autonomy software stack.

## 📝 Description

RViz configuration and launch files for the ARCS CoHORT project.
